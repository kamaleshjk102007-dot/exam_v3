Pipeline

How It Works

01
Connect Source

Select a classroom, choose Video or Real-Time mode. Upload a video file or enter your RTSP camera stream URL.

02
YOLOv8 Inference

FastAPI runs YOLOv8 on each frame in a thread pool. Ignored classes skipped immediately. Per-type confidence filters applied.

03
Stream to Browser

Annotated JPEG frames base64-encoded and pushed via WebSocket. Browser renders live feed with bounding boxes and HUD overlay.

04
Log & Alert

Every violation saved to MongoDB. Browser shows cooldown-controlled alerts. Snapshots saved with bounding boxes as evidence.
