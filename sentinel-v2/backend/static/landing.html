Built For
Real Exams

Every component is designed for exam hall conditions — multiple cameras, long sessions, live streaming and full alert history stored in MongoDB.

🎯
YOLOv8 Inference

Custom trained YOLOv8 model detects 5 violation classes per frame. Per-type confidence thresholds minimize false positives in crowded exam halls.

Real-Time AI

WebSocket Streaming

Annotated JPEG frames base64-encoded and pushed to the browser via WebSocket at up to 30fps. No polling. No refresh. Live bounding boxes.

Auto Snapshots

Violation frames saved with bounding boxes and HUD overlay. Per-type 10s cooldown prevents duplicate saves. View all snapshots in browser gallery.

Pipeline

How It
Works

01
Connect Source

Select a classroom, choose Video Mode or Real-Time. Upload a video file or enter your RTSP IP camera stream URL.

02
YOLOv8 Inference

FastAPI runs YOLOv8 on each frame in a thread pool. Ignored classes are skipped. Per-type confidence filters applied.

03
Stream to Browser

Annotated JPEG frames are base64-encoded and pushed via WebSocket. Browser renders live feed with bounding boxes and HUD overlay.

04
Log & Alert

Every violation saved to MongoDB. Browser shows cooldown-controlled alerts. Snapshots saved to disk with bounding boxes as evidence.
